Git Workflows

Make local, add Github remote

 
  mkdir <directory>
  cd <directory>
  git init 
  git remote add <initials> <repo-URL>
  git add . 
  git commit -m "..."
    For coauthoring: -m "...
    >
    >
    > Co-authored-by: <Co-author> <<github email>>"
  git push aps main

Clone from Github


  cd <parent-directory>
  git clone <repo-URL>

Branching & Merging


  git branch # see branches
  git checkout main
  git branch <new branch name>
  git checkout main
  git merge <branch to merge>
  git branch -D <obsolete branch>

Publish to GH-Pages


  git branch gh-pages
  git checkout gh-pages
  git push origin gh-pages
  Access via: <username>.github.io/<repo-name>

VS Code Shortcuts

  • Toggle sidebar: CTRL+B
  • Toggle terminal: CTRL+~
  • Open User Settings: CTRL+,
  • Quick Open: CTLR+p
  • Tab Through Files: CTRL+Tab (CTRL+Shift+Tab)
  • Open Current File in Separate Pane: CTRL+\
  • Quick Close: CTRL+W
  • Quick Replace: Highlight -> CTRL+H -> CTRL+Enter
  • Global Replace: CTRL+Shift+H
  • Command Palette: CTRL+Shift+P
  • Toggle Comment: CTRL+/
  • Skip to next word: CTRL+Left/Right
  • Highlight to next word: CTRL+Shift+Left/Right
  • Multiple cursors in vertical line: CTLR+Shift+Up/Down
  • Multiple Cursors: Alt+Click
  • Copy/Cut entire line: CTRL+C/CTRL+X (no need to select whole line)
  • Select Word: CTRL+D
  • Select All Instances of Word: CTRL+SHIFT+L
  • Rename variable: F2 with cursor on variable
  • Go to Function Definition: CTRL+Click
  • Toggle Zen Mode: CTRL+K Z
